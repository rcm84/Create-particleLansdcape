# Documentation

This folder contains lightweight starter-kit documentation for teams evaluating, integrating, and extending `ParticleLandscapeEngine`.

## Suggested reading order

1. [Project Overview](./project-overview.md)
2. [Examples](./examples.md)
3. [Assets Convention](./assets-convention.md)
4. [Changelog](../CHANGELOG.md)

## Why this folder exists

The repository is intentionally positioned as a downloadable starter kit rather than an npm-first library. These docs keep product-facing guidance separate from the API README so teams can onboard faster.
