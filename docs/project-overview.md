# Project Overview

`ParticleLandscapeEngine` is a premium starter kit for building stylized WebGL terrains from heightmaps using Three.js and GPU particles.

## Who this is for

- Creative developers building immersive landing pages, scene experiments, and portfolio environments.
- Product teams who want a polished terrain foundation without inventing a rendering stack from scratch.
- Agencies and studios packaging a reusable visual-system starter for multiple client projects.

## Common use cases

- Hero-section landscapes for marketing sites
- Endless terrain flyovers and sci-fi worlds
- Generative audio-reactive or data-reactive point-cloud environments
- Internal prototypes for terrain rendering, shaders, and interaction concepts

## What is included

- A modular engine under `engine/`
- Four runnable demos under `demos/`
- A Perlin-noise heightmap CLI under `terrain-tools/`
- A Vite development setup for local iteration
- Documentation for onboarding, assets, and examples

## Product positioning

This repository is best treated as a **downloadable toolkit**:

- clone it
- replace the sample terrain logic with project-specific scenes
- drop custom assets into the recommended folders
- evolve the engine architecture as your product needs mature
